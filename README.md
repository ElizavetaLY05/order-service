# Order Service Demo

Простой сервис для обработки заказов.

## Как запустить:

1. Установите PostgreSQL и NATS Streaming
2. Создайте базу данных
3. Запустите сервис: `go run main.go`
4. Отправьте заказ: `go run publish.go`
5. Откройте: http://localhost:8080
